# 📖 Overview

The **ATC Skill & Proficiency Management System** is a frontend web application developed to digitize the Air Traffic Control skill assessment and proficiency evaluation process.

The application provides an organized interface for managing assessment forms, recording trainee performance, tracking progress, and generating print-ready evaluation forms through a responsive web interface.

---

# ✨ Key Features

- 📋 Digital Skill Assessment Forms
- 📑 Proficiency Check Forms
- 📊 Automatic Score Calculation
- 🖨️ Print-Friendly A4 Layout
- 📱 Responsive Interface
- 🎯 Performance Evaluation
- 📂 Organized Form Management
