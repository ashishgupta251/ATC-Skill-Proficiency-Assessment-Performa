# ATC Skill & Proficiency Assessment Assessment Performa

A web-based frontend application developed to digitize the **Air Traffic Control (ATC) Skill Assessment** and **Proficiency Check** forms used during trainee evaluation. The project replicates the official assessment forms in a browser-based interface, making form filling, review, and printing easier while preserving the original layout.

## Project Overview

Traditionally, ATC Skill Assessment and Proficiency Check forms are maintained manually. This project provides a digital interface that closely follows the official form structure and enables users to complete assessments electronically.

The application has been designed with a strong focus on:

* Accurate form layout
* Easy navigation
* Automatic score calculation
* Print-ready output
* User-friendly interface

## Features

* Secure Login Interface (Frontend)
* Dashboard for Navigation
* Digital Skill Assessment Forms
* Digital Proficiency Check Forms
* Automatic Marks & Percentage Calculation
* Automatic Performance Result Evaluation
* Print-Friendly A4 Layout

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/ashishgupta251/ATC-Skill-Proficiency-Assessment-Performa.git
```

2. Open the project folder.

3. Launch `index.html` (or `login.html`) in any modern web browser.

No installation or server configuration is required.
